# Yuliang's Résumé

## Prerequisite

* Docker
* make

## Download the docker image and create the volume for MikTeX

``` bash
make init
```

## Update MikTeX packages when needed

``` bash
make update
```

## Make PDF

``` bash
make
```

## Clean

``` bash
make clean
make distclean
```
